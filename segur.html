Seguridad de la información

Añadir capas de seguridad y entender algunos ataques HTTP, también crear un scanner de puertos con Python

Express:

Framework (librería) para web hecho en NODE, uno de los más utilizados y más estables. Permite hacer una web, api res, difinir rutas y middlewares. Mucha documentación.

Instalación:requisitos (NodeJs y NPM) > npm install --save express

Middleware:

el cliente hace una petición al servidor URL y el middleware permite ejecutar un trozo de código (comprobaciones: autenticación, autorizaciones, cabeceras, obtener datos de sitios.. hay muchos y podemos crear los nuestros) antes de que se ejecute la acción a la base de datos.

HelmetJS

Un tipo de middleware para aplicaciones basadas en Express, ayuda a proteger configurando varios encabezados (headers) HTTP.
